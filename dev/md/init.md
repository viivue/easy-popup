## Init

Init ~~with~~ attribute `data-easy-popup` www.google.com

```html

<div data-easy-popup>
    <!-- Popup content -->
</div>
```

Or with JSON format for more options

```html
<!-- -->
<div data-easy-popup='{"id": "demo-1"}'>
```
